# Middleware 實作練習 
  藉由本專案練習，對 request 進來到 response 回去會經過一系列的流程，在這流程裡的程序，
  都叫做 middleware ， 此次練習目的為強化開發者對 middleware 的觀念。
## 功能

### 進度一

| 條件             | 結果                |
| -----------------|:------------------:|
| 時間戳計:(Taipei) |2022-03-22 10:46:42 |
| HTTP 的方法       | GET/POST           |
| URL              |  /, /new, /:id     |

  ####  **最後在伺服器要看到的訊息如 2022-03-22 10:46:51 | GET from /new**

 ## 開發環境

 - express: 4.17.3
